# AI Logs

This directory documents how AI tools were used during the development of Notely.

Per 8x Engineer submission guidelines, this folder captures the prompts, decisions, and debugging conversations that shaped the project.

- `02-build.md` — Build-time conversations: architecture, code generation, debugging
- `03-polish.md` — Polish-phase conversations: UI tweaks, copy, edge cases
